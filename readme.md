# Public Data Requests

This repo contains info about the public data requests I've made to government orgs.

I'm able to make these requests thanks to neat legislation like the Texas [Public Information Act](https://www.texasattorneygeneral.gov/open-government/office-attorney-general-and-public-information-act).
